kovra es **source-available** (código fuente disponible), publicado por **Kaeus Inc** bajo la
[Business Source License 1.1](https://mariadb.com/bsl11/) (BSL).

## Qué significa esto en la práctica

- **Tienes acceso al código fuente.** Léelo, compílalo, modifícalo y ejecútalo por tu cuenta.
- **Puedes usarlo** para desarrollo — la herramienta local, el CLI, el vault, el servidor
  MCP — que es exactamente lo que describe esta documentación.
- **Cada versión se vuelve completamente abierta** cuatro años después de su lanzamiento: en la
  fecha de cambio de esa versión se convierte a la **Apache License 2.0**.

La BSL existe para mantener el código fuente de kovra abierto e inspeccionable — algo importante
para una herramienta a la que confías tus secretos — al tiempo que le da al proyecto una base
sostenible. Lo único que restringe, hasta la fecha de cambio, es el derecho a ofrecer kovra como
un servicio comercial alojado que compita con el proyecto.

## Open-core

kovra-core — la herramienta local documentada aquí — es la parte source-available.
Las funcionalidades premium y organizacionales pueden residir en un componente separado bajo
términos distintos; nada en esta documentación depende de dicho componente.
