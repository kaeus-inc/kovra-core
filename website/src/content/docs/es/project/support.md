## Obtener ayuda

- Comienza con esta documentación — las páginas de [Preguntas frecuentes](/es/reference/faq/) y
  [Solución de problemas](/es/reference/troubleshooting/) cubren las dudas más comunes.
- Para errores e ideas de funcionalidades, abre un issue en
  [`kaeus-inc/kovra-core`](https://github.com/kaeus-inc/kovra-core/issues).

## Reportar un problema de seguridad

**Por favor, no reportes vulnerabilidades en un issue público.** Usa el sistema de
[reporte privado de vulnerabilidades](https://github.com/kaeus-inc/kovra-core/security)
de GitHub en el repositorio para que los mantenedores puedan atenderlo antes de que se divulgue.
Dado lo que kovra protege, esta es la contribución más valiosa que puedes hacer.

## Quién mantiene kovra

kovra es desarrollado y mantenido por **Kaeus Inc**. Es source-available
([Licencia](/es/project/license/)) y se desarrolla de manera abierta en GitHub.
