## Getting help

- Start with these docs — the [FAQ](/reference/faq/) and
  [Troubleshooting](/reference/troubleshooting/) cover the common questions.
- For bugs and feature ideas, open an issue on
  [`kaeus-inc/kovra-core`](https://github.com/kaeus-inc/kovra-core/issues).

## Reporting a security issue

**Please don't report vulnerabilities in a public issue.** Use GitHub's
[private vulnerability reporting](https://github.com/kaeus-inc/kovra-core/security)
on the repository so the
maintainers can address it before it's disclosed. Given what kovra guards, this is
the most valuable kind of contribution.

## Who maintains kovra

kovra is built and maintained by **Kaeus Inc**. It's source-available
([License](/project/license/)) and developed in the open on GitHub.
