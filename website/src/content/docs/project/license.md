kovra is **source-available**, published by **Kaeus Inc** under the
[Business Source License 1.1](https://mariadb.com/bsl11/) (BSL).

## What that means in practice

- **You get the source.** Read it, build it, modify it, and run it yourself.
- **You can use it** for development — the local tool, the CLI, the vault, the MCP
  server — which is exactly what these docs describe.
- **Each version becomes fully open** four years after its release: on that
  version's change date it converts to the **Apache License 2.0**.

The BSL exists to keep kovra's source open and inspectable — important for a tool
you trust with secrets — while giving the project a sustainable runway. The one
thing it withholds, until the change date, is the right to offer kovra as a
competing commercial hosted service.
