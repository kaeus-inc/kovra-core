# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and
personal attacks are not tolerated in any project space (issues, discussions,
reviews, email).

## Reporting

Report unacceptable behavior privately to **`info@kaeusanalytics.com`**. Reports
are handled confidentially by Kaeus Inc, the project maintainer.

The full text and enforcement guidelines are at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
